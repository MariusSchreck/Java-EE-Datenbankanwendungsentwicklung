Welcome to this Spring Boot Application

You can list all the advertisements here

You can search a specific advertisement under hs-kleinanzeigen/api/advertisements/{id}

Example hs-kleinanzeigen/api/advertisements/1

You can search a specific user under hs-kleinanzeigen/api/users/{id}

Example hs-kleinanzeigen/api/users/1

You can search a specific category under hs-kleinanzeigen/api/categories/{id}

Example hs-kleinanzeigen/api/categories/1

You can search a specific category under hs-kleinanzeigen/api/users/{userId}/notepad/

Example hs-kleinanzeigen/api/users/1/notepad/

The data is always returned in JSON format. To have a better view at the data you can install a JSON View Plugin in you Browser
